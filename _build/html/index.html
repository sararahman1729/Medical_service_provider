Module Documentation¶

Introduction¶

Baymax is a medical service provider chatbot imported in discord server.

Features¶

  1. Assess symptoms: It collects user symptoms and analyze them to give suggestions based on that.

  2. Collect patients’ data: It collects patients data for overall health concern.

  3. User feedback: User can give feedback to the chatbot and share his/her experience.

  4. Locate healthcare services: Baymax can give info about the healthcare services to the user

  5. Health tracking if he’s improving or not: Baymax can store the user info about their health

  6. Suggestions For emergency cases: Baymax can give suggestions in emergency cases

  7. Health advises for daily purposes: Baymax can give daily healthcare advices to the user

  8. Reminder or notifications for taking medicines: Baymax can give reminder for the medications to the user

Usage¶

Open your discord server and include the Baymax to your own server
